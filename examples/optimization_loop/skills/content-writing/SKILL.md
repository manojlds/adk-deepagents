# Content Writing Skill

## Blog Posts
- Use a compelling title with the primary keyword
- Hook readers in the introduction
- Structure with H2/H3 headers
- Include bullet points for scannability
- End with a call to action
- Aim for 500-1500 words

## Social Media
- Twitter/X: max 280 chars, use threads for longer content, 1-2 hashtags
- LinkedIn: professional tone, start with a hook, ~1300 chars
- Keep each post focused on one key message

## General
- Be concise and direct
- Use active voice
- Include specific examples and data
- Write for the target audience
